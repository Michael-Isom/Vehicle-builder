# Vehicle-builder
The Vehicle Builder Generator is a Node.js application designed to create and manage different types of vehicles, such as Trucks and Motorbikes.
This generator provides a flexible framework for defining various vehicle types, specifying their properties, and performing operations like towing, starting a CLI interface, and more.
